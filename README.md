# 🔥 IgnisShield / 安然火警 (Anran Fire Alarm)

> Replace traditional smoke detectors & CCTV. Real-time fire/smoke alert with AI vision. Low cost, high accuracy, easy deployment for smart communities.
>
> 替代传统烟雾报警器与监控摄像头的方案，基于 AI 视觉实现实时火灾/烟雾预警，为智慧社区提供低成本、高精度、易部署的解决方案。

## ✨ 项目亮点
- 🎯 **高精度识别**：基于轻量化 YOLO 模型，针对真实场景优化。
- 💰 **低成本部署**：无需昂贵硬件，普通摄像头 + 边缘设备即可运行。
- ⚡ **实时响应**：毫秒级检测，火灾初期即可预警。
- 🔓 **开源开放**：MIT 协议，自由使用与二次开发。
