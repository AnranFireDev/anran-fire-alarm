# 🔥 安燃无恙 / IgnisShield (Anran Fire Alarm)

> **中文**：替代传统烟感报警器与监控摄像头的方案，基于AI视觉实现实时火灾/烟雾预警，为智慧社区提供低成本、高精度、易部署的解决方案。
> **English**: A solution that replaces traditional smoke detectors and CCTV. Real-time fire/smoke warning based on AI vision, providing a low-cost, high-accuracy, and easy-to-deploy solution for smart communities.
---
## 🌟 项目亮点 / Project Highlights

- ✨ **高精度识别**：基于轻量化 YOLO-Fuse 多模态融合模型，针对真实场景全方位优化。
  **High Accuracy Detection**: Based on the lightweight YOLO-Fuse multi-modal fusion model, fully optimized for real-world scenarios.

- 🧨 **极低误报**：实测误报率≤2.5%，覆盖停车场、充电桩、小区等复杂场景。
  **Ultra-Low False Alarm Rate**: ≤2.5% in practical tests, covering complex scenarios such as parking lots, EV charging stations, and communities.

- 💰 **低成本部署**：无需昂贵硬件套件，普通摄像头 + 边缘设备即可运行。
  **Low-Cost Deployment**: No expensive hardware kits required, runs on a regular camera + edge device.

- ⚡ **实时响应**：毫秒级检测，火灾初期即可秒级预警。
  **Real-Time Response**: Millisecond-level detection, enabling early warning within seconds.

- 📄 **开源开放**：遵循 MIT 协议，自由使用与二次开发。
  **Open Source**: Licensed under MIT, free to use and modify for secondary development.
